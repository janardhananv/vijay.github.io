Creating a Configurable Onboarding Journey for Co-Brands.


The project involves solving for two major requirements.

1. To create an intuitive onboarding journey for credit line and card dispersal.
2. Creating easy to access configuration section for co-brands to manage the journeys.

Design Process


My contribution on this project was to create user flows, wireframes, mobile and web screens for onboarding and its configurations. Collaborators involved in the project are Product Manager, Product Head, Front End Team, Business Development team.

Started by understanding the onboarding flow of Hyperface’s existing clients to figure out features that are most common among them and the configurable elements.

Primary Flow


After multiple iterations on the low-fidelity wireframes, we came up with some basic designs for the mobile journey that could able to cover most the possible scenarios on the onboarding.

Upon approval, we worked on its corresponding web screens mostly replicating the mobile layout along with some additional elements which are configurable too.

Design Guidelines to Frontend


As the designs are configurable, we thought it is essential to educate the front end team with the design guidelines. So, we shared the designs with them along with guideline notes wherever it is required.

Designing the Configuration section


We then added a new section ‘Onboarding’ to the existing Hyperface dashboard, where Program Manager of a co-brand can configure onboarding journey and manage them.

We carefully curated this part, so that the user put minimal efforts for managing configurations. Some of the features includes reuse existing/default configs, preview for each steps and more.

Demo-able Prototype


As Hyperface developed this product, we created a figma prototype for the business development team to use them to demo it to the potential clients. Using this figma, business development can just change few configuration(logo, colors, brand name and more) in the figma to create prototype demo customized for the clients.

Closing Notes


The Project lasted around 4 months before the hand-off from the design. During the project I had opportunity to collaborate with different team in the organization to bring out the best solution for onboarding journey.

The Onboarding journey was first launched with Flipkart Wholesale partnering with LoanTap’s lending stack.
